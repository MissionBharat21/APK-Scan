# APK Sentinel

A terminal-based APK security tool for Linux and Termux with static analysis,
YARA signature scanning, manifest hardening checks, certificate reputation
tracking, post-install behavior monitoring, real-time intrusion detection,
and multi-channel alerting.

## Installation

```bash
git clone https://github.com/MissionBharat21/APK-Scan.git
cd APK-Scan
pip install -r requirements.txt
